## Развёртка сайта

1. Создайте venv через команду:

```bash
python -m venv env
source env/bin/activate
```

2. Установите зависимости

```bash
pip install -r requirements.txt
```

3. Запустите сайт

```bash
python flask-server.py
```

При необходимости можно отредактировать параметры в файле `config.json`
